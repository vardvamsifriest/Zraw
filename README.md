# Zraw - Real-Time Collaborative Whiteboard

🔗 **[Live Demo](https://zraw-web.vercel.app/)**

Real-time collaborative whiteboard supporting 20+ concurrent users.

## Features
- Real-time drawing with WebSocket
- Shapes, freehand, text tools
- Room-based collaboration
- Export to PNG/SVG

## Tech Stack
TypeScript, Next.js, React, Express, WebSocket, PostgreSQL, Prisma
